<head>


: the element is a container for metadata and is placed between the <html> tag and the <body> tag.

Code

<html>
<head>
<title>Title of the document</title>
</head>

<body<
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

Result

This is a heading

This is a paragraph.
